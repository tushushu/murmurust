## What
Python binding of MurmurHash3 Rust implementation.


## Requirements
-  Python: 3.8+
-  OS: Linux, MacOS and Windows


## Installation
Run `pip install mmr3`


## Examples
```Python
>>> import mmr3
# By default, seed=0, return unsigned int.
>>> mmr3.hash32('foo')
4138058784

# When seed = 100.
>>> mmr3.hash32('foo', seed=100)
3757588558

# Return signed int.
>>> mmr3.hash32('foo', signed=True)
-156908512
```
